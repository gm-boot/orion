# orion

### 项目说明

```text 
  orion项目主要功能是文档/文本/图片翻译, 在翻译文档同时保证原文档/图片格式不变,样式不变. 目前还在研发阶段预计4月底上线,敬请期待. 
```

### 主要功能描述

```text
  v 1.0 (2022-03-01)
  1. 翻译源
  2. 个性短语
  3. 行业词汇
  4. 敏感词
  5. OCR图片识别
  6. Tensorflow翻译词汇学习
  7. Word/Excel/Pdf 常用文档文件保留格式翻译
```
